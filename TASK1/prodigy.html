About Section

This navigation menu demonstrates several interactive features:

Services Section

The navigation uses modern CSS features like backdrop-filter for the blur effect and CSS transitions for smooth animations. The JavaScript handles scroll detection and mobile menu functionality.

Try hovering over the menu items to see the interactive effects, and scroll up and down to see how the navigation adapts to different scroll positions.

Portfolio Section

The menu maintains excellent contrast ratios in both light and dark states, ensuring accessibility while providing visual appeal. The hover effects include subtle transforms and color changes.

On mobile devices, the menu collapses into a hamburger menu that can be toggled to show the full navigation options.
